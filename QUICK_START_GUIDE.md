# クイックスタートガイド / Quick Start Guide

このガイドでは、Google Spreadsheetで株価取得関数を使い始める方法を簡単に説明します。

## 📋 5分でセットアップ

### ステップ 1: ファイルを開く

1. `GoogleAppsScript.js` ファイルを開く
2. 全ての内容をコピー (Ctrl+A → Ctrl+C)

### ステップ 2: Google Spreadsheetで設定

1. Google Spreadsheetを開く（新規または既存）
2. メニューから **「拡張機能」** → **「Apps Script」** をクリック
3. 表示されたエディタに、コピーした内容を貼り付け
4. **「保存」** ボタンをクリック（💾アイコン）
5. プロジェクト名を入力（例: "Stock Earnings Prices"）

### ステップ 3: 使ってみる

1. スプレッドシートに戻る
2. 任意のセルに以下を入力:

```
=getEarningsStockPrice("AAPL", 2023)
```

3. Enterキーを押す
4. 初回は権限の承認が必要です:
   - 「権限を確認」をクリック
   - アカウントを選択
   - 「詳細」→ 「安全ではないページに移動」をクリック
   - 「許可」をクリック

### ステップ 4: 結果を確認

数秒後、以下のような結果が表示されます:

```
┌─────────────────┬────────────────────────┐
│ Earnings Date   │ Stock Price (Close)    │
├─────────────────┼────────────────────────┤
│ 2023-11-02      │ 171.88                 │
│ 2023-08-03      │ 181.99                 │
│ 2023-05-04      │ 169.59                 │
│ 2023-02-02      │ 154.50                 │
└─────────────────┴────────────────────────┘
```

## 🎯 使用例

### 米国株
```
=getEarningsStockPrice("AAPL", 2023)    // Apple
=getEarningsStockPrice("MSFT", 2024)    // Microsoft
=getEarningsStockPrice("GOOGL", 2023)   // Google
```

### 日本株
```
=getEarningsStockPrice("7203.T", 2023)  // トヨタ自動車
=getEarningsStockPrice("9984.T", 2023)  // ソフトバンクグループ
=getEarningsStockPrice("6758.T", 2023)  // ソニーグループ
```

## ❓ よくある質問

### Q: エラーが表示される
**A:** 銘柄コードと年が正しいか確認してください。例: `"AAPL"`, `2023`

### Q: データが表示されない
**A:** 以下を確認:
- インターネット接続
- 銘柄コードが有効（Yahoo Financeで確認）
- その年に決算発表があったか

### Q: 日本株の銘柄コードは？
**A:** 証券コード + `.T` です。例: トヨタ自動車 = `"7203.T"`

## 🔧 トラブルシューティング

### 権限エラー
1. Apps Scriptエディタで「実行」→「関数を選択」→「getEarningsStockPrice」
2. 「実行」をクリックして手動で承認

### データが古い
1. セルを選択
2. Ctrl+Alt+Shift+F9 で強制再計算
